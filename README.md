## 🎯 סקירה כללית

מערכת מתקדמת ליצירת כרטיסי משחק אינטראקטיביים עם אפקטים ויזואליים מרשימים.
המערכת מספקת תבניות מוכנות לשימוש עבור משחקי קלפים פופולריים, כולל:

- אפקטים הולוגרפיים מתקדמים
- שכבות מרובות של אנימציות
- מערכת בקרת שכבות אינטראקטיבית
- תמיכה ב-RTL (עברית)

### 🌟 השראה

הפרויקט הזה מבוסס על תהליך היצירה המקצועי של כרטיסי פוקימון, שבו כל כרטיס עובר חודשים של עבודה ועיצוב שכבות.

## ✨ תכונות ושכבות

### שכבות האפקטים (לפי סדר מלמטה למעלה)

1. **שכבת רקע (Background Layer)** 🎨
   - גראדיאנטים מורכבים
   - מעברי צבע דינמיים
   - תבניות ייחודיות לכל סוג כרטיס

2. **שכבה הולוגרפית (Holographic Layer)** ✨
   - אפקט קשת בענן
   - תגובה לתנועת עכבר
   - מעברי צבע דינמיים

3. **שכבת קריסטל (Crystal Layer)** 💎
   - אפקט foil מתקדם
   - השתקפויות אור
   - זוהר פנימי

4. **שכבת קשת (Rainbow Layer)** 🌈
   - מעברי צבע רב-כיווניים
   - אנימציות gradient
   - אפקטי rainbow נדירים

5. **שכבת ניצוצות (Sparkle Layer)** ⭐
   - נקודות אור נוצצות
   - אנימציות אקראיות
   - אפקטי הבהוב

6. **שכבת תוכן (Content Layer)** 📝
   - טקסט ומידע
   - איקונים ואלמנטים
   - עיצוב ייחודי לכל סוג

7. **שכבת מסגרת (Border Glow)** 💫
   - זוהר חיצוני
   - אנימציות דינמיות
   - צבעים משתנים

### אפקטים נוספים

- **אפקט 3D** 🎭
  - סיבוב תלת מימדי
  - מעקב אחר עכבר
  - perspective דינמי

- **צללי טקסט** 🌑
  - עומק ויזואלי
  - ניגודיות משופרת
  - אפקטי glow

- **אנימציות קליק** 💥
  - התפוצצות ניצוצות
  - אפקטי feedback
  - אינטראקטיביות

## 📚 מדריכים

### מדריך 1: יצירת כרטיס פוקימון חדש

1. פתח את `pokemon/template.html`
2. העתק אחד מבלוקי ה-HTML של כרטיס קיים
3. שנה את התוכן (שם, HP, יכולות וכו')
4. התאם את `data-card-type` לסוג הרצוי
5. שמור וטען מחדש בדפדפן

### מדריך 2: התאמת צבעים לכרטיס דרגון בול

1. פתח `dragonball/dragonball-styles.css`
2. מצא את המשתנים:

   ```css
   .db-orange {
       background: linear-gradient(135deg, #ff6b35 0%, #ff8800 50%, #f77f00 100%);
   }
   ```

3. שנה את הצבעים לפי העדפתך
4. שמור ורענן

### מדריך 3: יצירת סוג כרטיס חדש

**שלב 1: הוסף רקע חדש ב-CSS**

```css
.my-new-type {
    background: linear-gradient(135deg, #color1, #color2, #color3);
}
```

**שלב 2: הוסף ב-HTML**

```html
<div class="card" data-card-type="my-new-type">
    <!-- תוכן הכרטיס -->
</div>
```

**שלב 3: הוסף אפקטים ייחודיים**

```css
.my-new-type .border-glow {
    box-shadow: 0 0 30px rgba(color, 0.6);
}
```

### מדריך 4: שימוש בתפריט השכבות לפיתוח

**תהליך מומלץ:**

1. **התחל עם "כבה הכל"** - כדי לראות את הבסיס
2. **הפעל שכבה אחת בכל פעם** - כדי להבין את השפעתה
3. **שנה קוד ובדוק** - עם השכבות הרלוונטיות מופעלות
4. **השתמש ב-"מינימלי"** - לבדיקת תוכן בלבד
5. **השתמש ב-"אפקטים בלבד"** - לבדיקת האפקטים

### מדריך 5: אופטימיזציה לביצועים

**טיפים לביצועים טובים יותר:**

1. **הפחת ניצוצות:** שנה את מספר הניצוצות בכרטיסים

   ```html
   <!-- במקום 7, השתמש ב-3-4 -->
   <div class="sparkle"></div>
   <div class="sparkle"></div>
   <div class="sparkle"></div>
   ```

2. **השתמש ב-will-change:** לאנימציות תכופות

   ```css
   .card-inner {
       will-change: transform;
   }
   ```

3. **הפחת שכבות:** במכשירים ניידים, כבה חלק מהאפקטים
