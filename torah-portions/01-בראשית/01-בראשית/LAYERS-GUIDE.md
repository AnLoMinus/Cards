## 🔧 ההבדלים טכניים

### visibility vs display

**🆕 שיטה חדשה (visibility):**

```css
.layer-off {
    visibility: hidden;
    opacity: 0;
}
```

✅ **יתרונות:**

- שומר על המקום של האלמנט
- לא משפיע על הזרימה
- מעבר חלק עם transition

**❌ שיטה ישנה (display):**

```css
.layer-off {
    display: none;
}
```

- מסיר את האלמנט לגמרי
- משנה את הזרימה
- אין מעבר חלק
